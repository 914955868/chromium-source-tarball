Chrome for Android has been updated to version 54.0.2840.85; the update will be available in Google Play over the next few days.  This release fixes some IME regressions discovered in the previous version as well as a number of security issues (this post will be updated with details when available).  A partial list of changes in this build can be found here.

If you find a new issue, please let us know by filing a bug. More information about Chrome for Android is available on the Chrome site.
