Good news, everyone!  Chrome 56 (56.0.2924.87) for Android has been released and will be available on Google Play over the course of the next week.  This release contains performance and stability fixes, as well as a few new features:
  • Quickly use emails, addresses, and phone numbers in web pages by tapping on them
  • Easily access downloaded files and web pages from the new tab page
  • Long press article suggestions on the new tab page to download them
A partial list of the changes in this build is available in the Git log.

If you find a new issue, please let us know by filing a bug. More information about Chrome for Android is available on the Chrome site.
